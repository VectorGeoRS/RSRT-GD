## 5. Road-Topology Masks

Road-topology masks used for geolocation are not provided as pre-generated files in the main RSRT-GD dataset. They are dynamically generated by clipping and rasterizing the shared road-vector reference according to anchor coordinates or candidate coordinates.

If users generate road-topology masks from the provided road-vector reference data, they must comply with the license and attribution requirements of the source road-vector data.

Road masks included in the auxiliary structure data, if provided, are intended only for supervising the structure-aware branch and should not be confused with pre-generated geolocation masks for the main dataset.
